# Marin Solutions - Blue Color Palette

## Color Template

This document defines the complete color palette for the Marin Solutions website, aligned with the blue beams background.

### Primary Colors
- **Primary Blue**: `#00b3ff` (Bright Cyan-Blue) - Main brand color, matches beams background light
- **Primary Dark**: `#0088cc` (Medium Blue) - Darker variant for gradients and hover states
- **Primary Light**: `#00d4ff` (Light Cyan) - Lighter variant for accents and highlights

### Secondary Colors
- **Secondary Blue**: `#0066cc` (Deep Blue) - Used for secondary elements and backgrounds
- **Secondary Dark**: `#004d99` (Darker Blue) - For depth and contrast
- **Secondary Light**: `#3399ff` (Sky Blue) - For lighter secondary elements

### Tertiary Colors
- **Tertiary Blue**: `#004080` (Navy Blue) - For depth and structure
- **Tertiary Dark**: `#003366` (Very Dark Blue) - For dark backgrounds
- **Tertiary Light**: `#66ccff` (Pale Blue) - For subtle highlights

### Complementary Colors
- **Complementary Orange**: `#ff6600` (Vibrant Orange) - For CTAs and important actions
- **Complementary Amber**: `#ffaa00` (Amber) - For warnings and highlights
- **Complementary Teal**: `#00cc99` (Teal) - Alternative accent color

### Neutral Colors
- **Background Dark**: `#0a1628` (Dark Blue-Black) - Main background, replacing #0a0a0a
- **Background Medium**: `#1a2a3a` (Medium Dark Blue) - For cards and elevated surfaces
- **Background Light**: `#2a3a4a` (Light Dark Blue) - For hover states
- **Text Primary**: `#FAFBFF` (Off-White) - Main text color
- **Text Secondary**: `rgba(250, 251, 255, 0.8)` - Secondary text
- **Text Tertiary**: `rgba(250, 251, 255, 0.6)` - Tertiary text
- **Border**: `rgba(0, 179, 255, 0.2)` - Blue-tinted borders
- **Border Light**: `rgba(0, 179, 255, 0.1)` - Subtle borders

### Usage Guidelines

#### Primary Blue (#00b3ff)
- Main brand color
- Buttons (primary)
- Links and hover states
- Icons and accents
- Active states

#### Secondary Blue (#0066cc)
- Secondary buttons
- Card backgrounds
- Section dividers
- Secondary accents

#### Tertiary Blue (#004080)
- Dark backgrounds
- Depth and shadows
- Structural elements

#### Complementary Orange (#ff6600)
- Call-to-action buttons
- Important highlights
- Success states
- Emphasis

#### Background Colors
- Use `#0a1628` for main page background
- Use `rgba(10, 22, 40, 0.6)` for card backgrounds with backdrop blur
- Use `rgba(10, 22, 40, 0.8)` for navbar and footer

### Gradient Examples
- **Primary Gradient**: `linear-gradient(135deg, #00b3ff 0%, #0088cc 100%)`
- **Secondary Gradient**: `linear-gradient(135deg, #0066cc 0%, #004080 100%)`
- **Hover Gradient**: `linear-gradient(135deg, #00d4ff 0%, #00b3ff 100%)`
- **CTA Gradient**: `linear-gradient(135deg, #ff6600 0%, #ff8800 100%)`

### Shadow Colors
- **Blue Glow**: `rgba(0, 179, 255, 0.3)` - For blue elements
- **Blue Glow Strong**: `rgba(0, 179, 255, 0.5)` - For emphasis
- **Dark Shadow**: `rgba(0, 0, 0, 0.4)` - For depth
